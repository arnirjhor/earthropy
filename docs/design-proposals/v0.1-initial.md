# v0.1 Initial Design Directions

Three distinct aesthetic directions for the apps/web landing page and apps/app shell. Pick one to advance to implementation, or pick best-of from each and document the synthesis. Logged into docs/design-patterns.md when decided.

The 17 SDG hex codes in `packages/sdg/src/sdgs.ts` are the categorical palette in every direction. The directions differ in the neutrals around them, the typography pair, the information density, and the rhetorical posture toward the visitor.

---

## Direction A — Field Record

**Mood:** Institutional. A working document, not a brochure. The platform takes itself as seriously as the work it hosts.

**Reference products:**
- **Bloomberg Terminal** — dense, monospaced, status-first; nothing decorative.
- **The New York Public Library catalog (nypl.org/research)** — formal serif headlines over a quiet utilitarian frame.
- **NASA Eyes (eyes.nasa.gov)** — mission-clock framing; data on top, chrome at the edges.
- **Hacker News** — boring on purpose; legibility over polish.
- **Information Is Beautiful** — categorical color used disciplined, not decorative.

**Typography:**
- Display: **GT Sectra** is closed-license — substitute **Source Serif 4** (Adobe, OFL) at 48/56 for landing H1 and 28/36 for section heads. A working serif with bookish authority; not literary, not academic-stuffy.
- Body: **IBM Plex Sans** (IBM, OFL) at 16/24. Designed for institutional UI; nine weights; broad multilingual coverage including Arabic (Plex Sans Arabic) and CJK companions.
- Mono (SDG codes, version strings, moderation IDs, timestamps): **IBM Plex Mono** at 13/20.
- Why this pair: Plex was commissioned to look like a serious organization talking, not a startup. Pairing with Source Serif elevates landing-page headlines without losing the operational tone. The same family covers Arabic, Devanagari, Thai, Hebrew, Cyrillic, Greek — every v0.1 locale except Swahili (Latin coverage is sufficient there).

**Color system:**
- Neutrals (light): background `#F7F6F2` (warm paper, not stark white), surface `#FFFFFF`, surface-elevated `#FFFFFF` with a 1px `#1A1A1A` 10%-alpha border, border `#1A1A1A` 16%-alpha (`#E1DFD9` effective), text `#0F0F0F`, text-muted `#4A4A4A`.
- SDG color usage: SDG colors appear as **4px left rules** on cards, as **the chip background** behind the SDG number in a square mono-set chip (e.g. a red `#E5243B` 24px square with white "01"), and as **typographic accent** on the SDG hub headline (the goal name set in its own color at large size). Body text never sits on a saturated SDG fill — chips and rules only. AAA contrast (#0F0F0F on `#F7F6F2` is ~17:1).
- Dark mode: background `#0E0F11`, surface `#16181B`, border `#FFFFFF` 14%-alpha, text `#F2F1ED`, text-muted `#A6A39C`. SDG colors are not desaturated — they remain their official hex. SDG 7 (`#FCC30B`) and SDG 5 (`#FF3A21`) gain AAA contrast against the dark surface; SDG 17 (`#19486A`) drops to AA-large only — use it only as a 4px rule or on a light "onColor" badge.

**Motion treatment:** Spare. 120ms fade-in on route change, 180ms ease-out on hover state. No parallax, no scroll-jacking, no entrance choreography. `prefers-reduced-motion`: all transitions drop to 0ms; state changes happen instantly.

**Information architecture:**
- Landing page (apps/web):
  ```
  ┌────────────────────────────────────────────────────┐
  │ EARTHROPY                       en  ar  es  +6  ↪ │
  ├────────────────────────────────────────────────────┤
  │                                                    │
  │  A neutral place to coordinate work                │
  │  on the 17 UN Sustainable Development Goals.       │
  │                                                    │
  │  [ Sign up — free, forever ]   [ Read the docs ]   │
  │                                                    │
  ├────────────────────────────────────────────────────┤
  │ 01 ▌No Poverty     02 ▌Zero Hunger    03 ▌Health   │
  │ 04 ▌Education      05 ▌Equality       06 ▌Water    │
  │ … (full 17-cell grid, each cell links to its hub)  │
  ├────────────────────────────────────────────────────┤
  │ How it works  /  Moderation  /  Self-host  /  Code │
  └────────────────────────────────────────────────────┘
  ```
  Sections: (1) headline + dual CTA; (2) the 17-cell SDG grid as the proof and the navigation; (3) three-column "what this is / how it works / what it isn't"; (4) footer with AGPL badge, repo link, governance link.
- Authenticated shell (apps/app):
  ```
  ┌─[ EARTHROPY ]──[ search ]───────[ ◐ EN ] [ ? ] [ ◯ ]
  │ ▌Feed                                              │
  │ ▌Groups (4)                ┌───────────────────┐  │
  │ ▌SDGs (7)                  │  Personal feed    │  │
  │ ─────                      │  post · post · post│  │
  │ 01 ▌No Poverty             │                   │  │
  │ 13 ▌Climate Action         │                   │  │
  │ ...                        └───────────────────┘  │
  ```
  Left rail: persistent. Breadcrumb is text on a single line: `Feed / Climate Action / Group: Coastal Erosion Atlas / Post #1429`. Default page after login = personal feed (groups joined + SDGs followed, reverse-chronological, no algorithmic re-ordering).
- Group page: header bar with group name, SDG chips, member count, role indicator; tab strip (Posts / Members / About); post list as a 1-up column with 4px SDG rule on the left of each post card.
- Post detail page: title in display serif, body in body sans, mono "Status: published · moderated by ai-v0.1 · 2026-05-18 14:22 UTC" line directly under the byline. Comments thread with 1px indentation rules. A moderation status pill anchored to the upper right of the post body shows `published`, `pending review`, etc. — pill color uses neutrals plus a single dot in the SDG color of the post's primary tag.

**RTL behavior:** Layout mirrors fully. The 4px SDG rule moves from `border-left` to `border-right`. The mono SDG number chip stays LTR for digit ordering (numbers are still read left-to-right within Arabic text per Unicode bidi). Icons that imply direction (arrows, breadcrumb separators) flip; the SDG icons themselves and the Earthropy logotype do not. Plex Sans Arabic has been tested with Plex Sans Latin at the same x-height; mixed-script lines balance.

**Accessibility:**
- Contrast: Body text (`#0F0F0F` on `#F7F6F2`) ~17:1 AAA. Muted text (`#4A4A4A` on `#F7F6F2`) ~8.6:1 AAA. SDG 7 (`#FCC30B`) and SDG 12 (`#BF8B2E`) require black `onColor` (already declared in `sdgs.ts`) for AAA on chip text. Dark mode: text on background ~16:1 AAA.
- Keyboard nav: 2px solid focus ring in `#0F0F0F` (light) / `#F2F1ED` (dark) with 2px offset; visible against every SDG color. Tab order follows DOM; skip-links to "main content" and "primary navigation" at the top of every page.
- Screen reader: Landmark structure is `<header>`, `<nav aria-label="primary">`, `<main>`, `<aside aria-label="filters">`, `<footer>`. SDG chips render as `<span aria-label="SDG 13: Climate Action">` with the colored block as `aria-hidden` and the number/name as the readable text.
- Low-bandwidth: First paint is HTML + 14KB critical CSS inlined + no fonts (FOUT with `font-display: swap`). Plex weights load deferred; system serif/sans fallbacks declared in the font stack. No JS required to read any post, group, or SDG hub. Images use `loading="lazy"` and width/height attributes.

**Trade-offs:** Strongest "this is serious infrastructure" read. Risk: reads as cold or bureaucratic to a casual visitor who came from social media. Lowest distinctiveness if reduced to a sketch — its power is in the discipline of execution, not in a hero gesture.

**What it would look like in one paragraph:** Imagine the front page of a UN statistical yearbook redrawn for the web. Warm paper-cream background, deep near-black text, a calm Source Serif headline that says exactly what the platform is in one sentence. Below: a clean 17-cell grid where each cell is the SDG number in monospace, a 4px vertical bar in the goal's official color, and the goal's name in IBM Plex. No photography, no illustration, no gradient, no glow. Inside the app, a left rail of joined groups and followed SDGs sits beside a single-column reverse-chronological feed of post cards, each carrying its SDG colors as edge rules. It looks like a working tool that respects your time.

## Direction B — Civic Atlas

**Mood:** Cartographic. The platform as a quiet, legible map of the work — coordinates, scales, legends. Information-dense without being noisy.

**Reference products:**
- **OpenStreetMap (openstreetmap.org)** — utilitarian map chrome, layered legend, public-good tone.
- **Are.na** — generous whitespace, content-first, no decorative chrome; channels-as-collections idea ports to groups.
- **Stamen Design's Toner / Watercolor map styles** — restrained palette discipline, a single category accent.
- **The MoMA collection site (moma.org/collection)** — labels-as-design, museum-card metadata aesthetic.
- **Field Notes notebook covers (fieldnotesbrand.com)** — small caps + dense metadata + grid paper texture done with discipline.

**Typography:**
- Display: **Fraunces** (Undercase Type, OFL) at 56/60 for hero; an "old-style serif with new ideas," parametrically expressive but readable. Use at low optical-size settings (`opsz`) and modest weight (450) to avoid theatricality. Italic carries quotation and emphasis at the same weight.
- Body: **Public Sans** (US Web Design System / GSA, OFL) at 16/24. Designed for government/civic interfaces; a Libre Franklin descendant; calm, neutral, no decorative letterforms.
- Mono: **JetBrains Mono** (JetBrains, OFL) at 13/20 for SDG numbers, coordinates, IDs, dates.
- Why this pair: Fraunces gives the landing page one moment of warmth (the headline) without slipping into branding-deck territory; Public Sans is the workhorse with civic associations; JetBrains Mono is the most legible open monospace at small sizes and ships full Latin + Cyrillic + Greek (Arabic is composed in Plex Sans Arabic as the fallback). Fraunces and Public Sans have nearly identical x-heights — they nest cleanly.

**Color system:**
- Neutrals (light): background `#FAFAF7` (very pale linen), surface `#FFFFFF`, surface-elevated `#FFFFFF` with `#1F1F1F` 8%-alpha shadow (1px y-offset, no blur — a hairline lift, no glassmorphism), border `#1F1F1F` 12%-alpha, text `#111111`, text-muted `#535353`. A single accent for non-SDG UI affordances (links, focus, primary buttons): a flat **`#0A4D6C`** (sea-chart blue, distinct from SDG 14 `#0A97D9` and SDG 16 `#00689D` by being darker and less saturated — confirmed AAA against background).
- SDG color usage: SDG colors are **map-legend swatches** — small (12px) square chips paired with the SDG number, used as filters, tags, and category indicators. On a group's "atlas card" the primary SDG appears as a **6px top stripe**; secondary SDGs appear as 12px swatches in the metadata strip. Posts in feed are not flooded with color — only the chip and the stripe carry it. A "currently viewing" SDG (when filtering) draws an 8px outer frame around the post list in that color.
- Dark mode: background `#101418`, surface `#171C21`, border `#FFFFFF` 12%-alpha, text `#F0EFEA`, text-muted `#9C9890`, accent `#7BB8D6` (the sea-chart blue lifted for contrast). SDG colors hold; SDG 7 yellow becomes the highest-contrast tag (good — it's used for "Energy").

**Motion treatment:** Cartographic — slow, considered. 220ms ease-in-out for panel transitions (drawer opening, filter applying); 80ms snap for buttons. Map-style zoom transitions on the SDG hub (the SDG number scales from chip-size to display-size with a 280ms cubic-bezier(0.2, 0.8, 0.2, 1)). `prefers-reduced-motion`: scale transitions become cross-fades at 100ms.

**Information architecture:**
- Landing page (apps/web):
  ```
  ┌────────────────────────────────────────────────────┐
  │ Earthropy                          [Sign in] [Join]│
  ├────────────────────────────────────────────────────┤
  │                                                    │
  │            A map of the work that matters.         │
  │            17 goals. One platform. Forever free.   │
  │                                                    │
  │                  [ Start a group ]                 │
  │                                                    │
  ├──── The 17 goals ──────────────────────────────────┤
  │ ▣01 No Poverty     ▣02 Zero Hunger   ▣03 Health   │
  │ ▣04 Education      ▣05 Equality      ▣06 Water    │
  │ … 17 swatch-cells, each linking to its SDG hub …   │
  ├──── What's on the map today ───────────────────────┤
  │ • 142 groups across 47 countries                   │
  │ • 1,820 posts in the last 30 days                  │
  │ • 9 languages (more in progress — help translate)  │
  ├────────────────────────────────────────────────────┤
  │ Read the moderation policy · Self-host · Source    │
  └────────────────────────────────────────────────────┘
  ```
  Sections: (1) one-line headline + single CTA; (2) the 17-goal "legend" as the primary navigation; (3) running stats ("what's on the map today") as honest, low-key proof; (4) footer.
- Authenticated shell (apps/app):
  ```
  ┌──────────────────────────────────────────────────────┐
  │ Earthropy   /coastal-erosion-atlas    [ search  ]  ◐ │
  ├──────────┬───────────────────────────────────────────┤
  │ FEED     │ ▣06 ▣11 ▣14   Coastal Erosion Atlas      │
  │ GROUPS   │ ─────────────────────────────────────     │
  │ ▣01 ...  │ Post · Sea-wall data from Cox's Bazar     │
  │ ▣13 ...  │ Post · Volunteer mapping weekend          │
  │ ─────    │ Post · …                                  │
  │ ATLAS    │                                           │
  │ MOD QUE  │                                           │
  └──────────┴───────────────────────────────────────────┘
  ```
  Left rail is a "legend" with two sections: Joined Groups (with primary SDG swatches inline) and Followed SDGs. Breadcrumb is map-coordinate styled in mono: `feed › ▣13 ›  /coastal-erosion-atlas › post #1429`. Default landing = personal feed.
- Group page: top is a wide "atlas card" — group name in Fraunces, location free-text + preferred locale in mono, SDG swatches as legend. Tabs underneath. Post list reads as cartographer's notes.
- Post detail page: a full-width single column; mono coordinates strip ("Posted 2026-05-18 · /coastal-erosion-atlas · author: handle@earthropy · ▣11 ▣14"); body in Public Sans 17/27; comments as a threaded list with the same coordinates strip pattern.

**RTL behavior:** Rail mirrors to the right. Swatches stay as squares (orientation-neutral). The 6px top stripe stays on top. Mono coordinates in the breadcrumb read RTL at the line level but the SDG number chips inside read LTR per Unicode bidi rules. Fraunces does not ship Arabic — Arabic display headlines fall back to **Reem Kufi** (Khaled Hosny, OFL) which carries similar "considered, slightly literary" tone; Public Sans falls back to **IBM Plex Sans Arabic** for body.

**Accessibility:**
- Contrast: `#111111` on `#FAFAF7` ~17:1 AAA. Muted `#535353` on `#FAFAF7` ~7.6:1 AAA. The `#0A4D6C` accent on `#FAFAF7` ~9.8:1 AAA. SDG 7 (`#FCC30B`) chip uses its declared `onColor: #000000` for the digits — AAA. SDG 12 (`#BF8B2E`) on `#FAFAF7` is ~4.1:1 (AA-large only) — restricted to swatches sized ≥24px and never used for body text.
- Keyboard nav: 2px focus ring in `#0A4D6C` (light) / `#7BB8D6` (dark). Focus moves through landmarks via skip-links.
- Screen reader: Each SDG swatch is `<span role="img" aria-label="SDG 6: Clean Water and Sanitation">`. Atlas cards announce as group with primary SDG named first.
- Low-bandwidth: 17 SDG icon SVGs are inlined in the critical bundle (each ~600 bytes optimized — ~10KB total). The "atlas card" needs no images. Maps (if/when added — out of v0.1 scope) would use static raster tiles with text fallback.

**Trade-offs:** Most distinctive front door of the three. Risk: the "atlas" metaphor over-promises if the platform isn't actually map-centered in v0.1. Mitigation: lean on the legend/swatch language rather than literal maps; defer geographic mapping to v0.2.

**What it would look like in one paragraph:** Imagine an OpenStreetMap with the noise stripped out. Linen-pale background, a single warm-serif sentence that names the mission, and a clean 17-square legend underneath where each square is the goal's official color. Inside the app, the left rail reads like a map legend — joined groups with their primary SDG swatch inline, followed SDGs as their own list — and the main content is a quiet column of post cards each carrying a 6px top stripe in its primary SDG color and a mono "coordinates" line of metadata. The whole thing feels like a public-good cartography project: nothing decorative, everything labeled, the work itself is the foreground.

## Direction C — Press Sheet

**Mood:** A printed broadside. A printer's mark, a masthead, a hierarchy you can scan from across the room. The platform announces work, not features.

**Reference products:**
- **The Public Domain Review (publicdomainreview.org)** — broadside-newspaper masthead, dense but humane.
- **Increment magazine (web archive)** — wide measure, generous leading, two-column reading on long form.
- **Linear circa 2020** — keyboard-first chrome, dark surfaces with a single accent, taut typographic hierarchy.
- **The Long Now Foundation (longnow.org)** — centuries-scale gravity, restrained palette.
- **Letterform Archive (letterformarchive.org)** — typographic restraint as the design.

**Typography:**
- Display: **Newsreader** (Production Type, OFL) at 64/64 for the landing masthead. A variable serif designed explicitly for headlines + long-form on screen; high-contrast at display sizes, gentle at body.
- Body: **Inter Tight** is disallowed (Inter family banned). Substitute: **Manrope** (Mikhail Sharanda, OFL) at 16/26. Geometric humanist sans with a distinct (not generic) lowercase 'a' and 'g'; wider apertures than Inter; eight weights. Or alternative: **Söhne** is closed-license — second choice is **Geist Sans** (Vercel, OFL) — but Geist is currently associated with one company; Manrope is the safer self-host pick.
- Mono: **Geist Mono** (Vercel, OFL) at 13/20 — Manrope-pairs well via shared geometry. Fallback: **JetBrains Mono**.
- Why this pair: Newsreader is unmistakably a broadsheet serif; Manrope is calm, modern, distinct from system-ui in its open-aperture letterforms. The pair has Latin Extended + Cyrillic; Arabic falls back to **Cairo** (Mohamed Gaber, OFL) for body and **Lateef** (SIL, OFL) for display — Cairo's geometric construction sympathetic to Manrope, Lateef's Naskh weight comparable to Newsreader.

**Color system:**
- Neutrals (default = dark, with a light variant): background `#0B0C0E` (near-black, not pure black to avoid screen burn on OLED at the edge), surface `#14161A`, surface-elevated `#1B1E23`, border `#FFFFFF` 12%-alpha, text `#F2F1ED` (cream-white, not pure white — easier on eyes for long reads), text-muted `#A6A39C`. Light variant: background `#F2F1ED`, surface `#FFFFFF`, text `#0B0C0E`, text-muted `#5A5854`.
- SDG color usage: SDG colors are **the only saturated color** anywhere — they appear as **the SDG number in display type**, set in the official color at the top of every SDG hub and as a column-marker (12px square + number) on each post in feed. Group pages and the masthead use a "primary SDG color" as a single-strip accent above the group title (a 2px hairline that runs the width of the column). The landing page masthead has the 17 SDG colors set as a 17-cell color-bar across the top of the page — the platform's flag.
- Dark mode is the default; SDG colors look like neon ink on newsprint when set on `#0B0C0E`. SDG 8 (`#A21942`) and SDG 17 (`#19486A`) become moody and serious — appropriate for "Decent Work" and "Partnerships." SDG 7 (`#FCC30B`) is a strong yellow against the dark — almost a highlighter.

**Motion treatment:** Editorial. Page transitions = 200ms ease-out cross-fade. Reading content has no motion. The masthead's 17-cell color bar has a 600ms entrance on first load (each cell fades in at +30ms intervals from left, RTL-aware: from right when `dir="rtl"`) — used once per session to introduce the palette, then static. `prefers-reduced-motion`: the bar simply appears.

**Information architecture:**
- Landing page (apps/web):
  ```
  ┌────────────────────────────────────────────────────┐
  │ ██████████████████  ← 17 SDG color cells, full bleed
  ├────────────────────────────────────────────────────┤
  │ EARTHROPY                          NO. 001 · 2026  │
  │ ────────────────────────────────────────────────── │
  │                                                    │
  │              Earth + Entropy.                      │
  │              Action against the                    │
  │              planet's growing disorder.            │
  │                                                    │
  │                                                    │
  │     A neutral, forever-free platform to            │
  │     coordinate work on the 17 UN Sustainable       │
  │     Development Goals.                             │
  │                                                    │
  │             [ Join — it's free, always ]           │
  │                                                    │
  ├──── THE 17 GOALS ──────────────────────────────────┤
  │ 01  No Poverty                                     │
  │ 02  Zero Hunger                                    │
  │ 03  Good Health and Well-Being                     │
  │ ...  (full list, hover/focus reveals description)  │
  ├──── PRINCIPLES ────────────────────────────────────┤
  │ Forever free.                                      │
  │ Corporation-agnostic.                              │
  │ Universally accessible.                            │
  │ Transparent moderation.                            │
  ├────────────────────────────────────────────────────┤
  │ AGPL-3.0 · Source · Docs · Code of Conduct         │
  └────────────────────────────────────────────────────┘
  ```
  Sections: (1) 17-cell color bar = masthead flag; (2) headline + tagline in Newsreader; (3) the 17 goals as a single-column reading list (not a grid — a manifesto); (4) principles as a four-line credo; (5) footer.
- Authenticated shell (apps/app):
  ```
  ┌──────────────────────────────────────────────────────┐
  │ ██████████████████  (8px tall 17-cell color bar)     │
  ├──────────────────────────────────────────────────────┤
  │ EARTHROPY      Feed  Groups  SDGs  Mod  [⌘K] [◐] ◯  │
  ├──────────────────────────────────────────────────────┤
  │                                                      │
  │   01 ┃ Post title in Newsreader, two columns        │
  │      ┃ of body text in Manrope below                │
  │      ┃ — handle@earthropy · 2h ago · ▌13 ▌07        │
  │                                                      │
  │   13 ┃ Another post …                                │
  │                                                      │
  └──────────────────────────────────────────────────────┘
  ```
  Top nav, not left rail (the broadsheet wants the page width). A persistent 8px-tall 17-cell color bar runs across the top of every page as a navigation device — clicking any cell goes to that SDG hub. Command-K opens the universal search/jump (groups, SDGs, members). Breadcrumb is implicit in the masthead style of each page rather than a chevron path. Default landing = personal feed in single-column broadsheet layout (no sidebar in feed view — the focus is reading).
- Group page: masthead-style — group name as the title set in Newsreader, the group's primary SDG as a 2px hairline above, SDG chip strip below the title, then tabs (Posts / Members / About), then the post list rendered as a column of dated headlines.
- Post detail page: title set in Newsreader at display size, byline in mono, body in Manrope 17/28 at a 68ch measure (one column, wide whitespace gutters). Comments below the body separated by a horizontal hairline; each comment is a smaller masthead unit (author + time + body + moderation pill).

**RTL behavior:** Top nav mirrors. The 17-cell color bar starts at the right for `dir="rtl"` and reads SDG 1 → 17 from right to left (which is correct — Arabic readers expect "first" on the right). Numerals in post columns stay LTR. Newsreader has Latin coverage only; Arabic display uses Lateef. Manrope's Arabic fallback is Cairo — both have generous counters so the broadsheet feel survives translation. The hairline column rule moves from left to right of the column number.

**Accessibility:**
- Contrast: Dark mode (default) — `#F2F1ED` on `#0B0C0E` ~17.5:1 AAA. Muted `#A6A39C` on `#0B0C0E` ~8.4:1 AAA. SDG 7 (`#FCC30B`) on `#0B0C0E` ~13.2:1 AAA — display-size numerals readable from a meter away. SDG 17 (`#19486A`) on `#0B0C0E` ~3.6:1 (AA-large only) — restricted to display-size numerals (≥24px) and never used for body. Light mode: `#0B0C0E` on `#F2F1ED` ~17.5:1 AAA.
- Keyboard nav: 2px outline-offset focus ring in `#F2F1ED` (dark) / `#0B0C0E` (light) with 2px offset. Top nav has roving tabindex; Command-K is documented in `?` help overlay.
- Screen reader: The 17-cell color bar is `<nav aria-label="Sustainable Development Goals">` with `<a aria-label="SDG 1: No Poverty">` per cell; the colored fills are `aria-hidden`. The masthead "NO. 001 · 2026" is decorative (`aria-hidden`).
- Low-bandwidth: Dark mode default = fewer cycles on OLED + smaller perceived contrast brightness on mobile. Critical CSS inlined; Newsreader and Manrope variable fonts loaded once with `font-display: swap`; mono loaded only on routes that show it (the dashboard, post pages). No-JS read works fully — comments render server-side; submit forms degrade to a POST.

**Trade-offs:** Highest visual ambition — most "this looks like something" of the three. Risk: the broadsheet metaphor leans literary/Western; international maintainers may find it less universal than Direction A or B. The Newsreader + Manrope pair carries more "magazine" character, which some NGO audiences may read as "design-y" rather than "operational." Hardest to keep tonally right as features grow.

**What it would look like in one paragraph:** Imagine a broadsheet newspaper masthead reimagined for the climate era. Across the top of every page runs a thin 17-cell band in the official SDG colors — a flag. Below it, "EARTHROPY" set in calm serif caps with a publication-style "NO. 001 · 2026" line, then a one-sentence Newsreader headline in cream on near-black: "Earth + Entropy. Action against the planet's growing disorder." The whole landing page reads top-to-bottom in a single editorial column. Inside the app, posts arrive in feed as headlines with a 2-digit SDG number column and a 2px vertical hairline in the goal color, body set wide and quiet. It feels like reading a serious paper run by people who refuse to write filler.

## Comparison table

| Axis | A — Field Record | B — Civic Atlas | C — Press Sheet |
|---|---|---|---|
| Tone | Institutional, terminal-adjacent, unornamented | Cartographic, civic, public-good utility | Editorial, masthead, gravity-by-restraint |
| Risk | Reads as cold or bureaucratic; least visual character | "Atlas" metaphor over-promises if features don't deliver | Broadsheet may read as Western-literary; tonally hardest to scale |
| Time to ship | Fastest — fewest custom typography moments, simplest motion | Moderate — needs disciplined swatch system + accent color | Slowest — wider measure, broadsheet hierarchy needs typographer-grade care; 17-cell flag is load-bearing visual |
| Distinctive? | Distinctive through discipline only — easy to underestimate from a sketch | Most distinctive at first impression; clear visual idea | Boldest masthead moment; most ownable as a brand |

## My recommendation

**Direction A — Field Record** is the right v0.1 default. It is the lowest-risk read of "this is serious infrastructure," it ships fastest, IBM Plex's multilingual family covers every v0.1 locale without fallback gymnastics, and it cedes the spotlight to the SDG palette rather than competing with it. The audience (NGOs, researchers, local communities) is the audience IBM Plex was drawn for; the tonal match is exact. The dark-mode and AAA contrast story is the cleanest of the three.

What to borrow:
- From **B (Civic Atlas)**: the 12px square swatch + SDG number as the canonical chip pattern across the platform, and the "atlas card" layout for group pages. These read as more legible than rule-only treatment when many SDGs are tagged. Also borrow the explicit civic accent color (`#0A4D6C`) for primary action buttons — Direction A's plain near-black buttons can feel dour, and the sea-chart blue gives the platform one non-SDG action color that doesn't compete with the 17.
- From **C (Press Sheet)**: the 8px-tall 17-cell color bar as the persistent top-of-page navigation device, used in apps/app only (not on the marketing landing). This solves the "where do I jump to SDG N?" navigation problem with one always-visible affordance and gives the platform a small piece of memorable visual identity. Set the bar 4px tall instead of 8px in the authenticated shell to keep the gravity of Field Record intact.
